# Уебинар фуния „AI Машината за Клиенти" — архитектура и пускане

Построена 2026-07-06. Модел: безплатен Zoom уебинар (като masterclass.bg/2026, но по-силен) →
курс 500 € → менторска програма 2000 € (ъпгрейд с кредит).

## Архитектура (какво къде живее)

```
РЕКЛАМА (Meta) ──► /webinar (лендинг + форма)
                       │  POST /api/webinar/register
                       │  → CRM контакт (source: webinar) + activity
                       │  → имейл: потвърждение + подарък + Zoom линк
                       ▼
                /webinar/registriran (thank-you)
                       │  сваля /api/webinar/podarak (PDF „AI Стартов Пакет")
                       ▼
                ZOOM УЕБИНАР (на живо, 90 мин)
                       │  в края: оферта курс 397 € (уебинар цена, иначе 500 €)
                       ▼
                /kurs (продажбена страница)
                       │  CheckoutButton → POST /api/checkout → Stripe Checkout
                       ▼
                /kurs/uspeh (успех + ЕДНОКРАТЕН ЪПСЕЛ)
                       │  менторство: 1500 € вместо 2000 € (кредит от курса)
                       │  CheckoutButton product=mentorship-upgrade
                       ▼
                Stripe webhook /api/webhooks/stripe
                       → CRM: контакт stage=client + activity + плащане (dedupe)
                       → имейл известие към админа
```

## Офертна стълба

1. **Безплатно**: уебинар + „AI Стартов Пакет" PDF (стойност 90 €) — събира имейли.
2. **Курс „AI Продажбена Машина"** — 500 € (на уебинара: 397 €, обявява се на живо).
3. **Менторска програма 1-на-1, 4 месеца** — 2000 €; купилите курса доплащат 1500 €
   (кредитът се показва само на /kurs/uspeh — класически one-time-offer).

Всички цени се сменят на ЕДНО място: `lib/webinar/config.ts` → `OFFERS`.

## Какво остава да настроиш ти (по ред)

1. **Дата и час** — в `lib/webinar/config.ts` сложи `dateISO` (пример:
   `"2026-07-21T19:00:00+03:00"`). Страницата автоматично показва датата + обратно броене;
   имейлите я включват сами. Препоръка: вторник/сряда 19:00, 10–14 дни след старта на рекламата.
2. **Zoom** — създай миитинг (Zoom → Schedule → Webinar или Meeting, регистрация изключена,
   waiting room изключен) и сложи линка в `zoomJoinUrl`. Той се праща само по имейл, не се показва публично.
3. **Stripe** — в dashboard.stripe.com:
   - Developers → API keys → копирай **Secret key** → Vercel env `STRIPE_SECRET_KEY`
   - Developers → Webhooks → Add endpoint `https://promarketing.pw/api/webhooks/stripe`,
     event `checkout.session.completed` → копирай **Signing secret** → env `STRIPE_WEBHOOK_SECRET`
   - Без тези 2 ключа бутоните за плащане показват учтиво съобщение (нищо не се чупи).
4. **Реклама (Meta)** — кампания Lead/Traffic към `/webinar`. 3 ъгъла × 10–20 €/ден:
   а) „спираш да губиш запитвания", б) „конкурентите ти вече ползват AI", в) „подаръкът" (пакетът за 90 €).
5. **Имейл поредица преди уебинара** (пуска се от CRM-а/Resend):
   - Ден 0: потвърждение + подарък (автоматично, вече работи)
   - Ден -3: „какво ще видиш" + мини урок
   - Ден -1: напомняне + календарен линк
   - Час -1: „започваме след час" + Zoom линк
   - След края: replay 48ч (по избор) + оферта курс 397 € с краен срок

## Скрипт на самия уебинар (90 мин, тествана структура)

1. **0–10 мин** Кука: жива система с реални числа (демото /demo на екран).
2. **10–60 мин** 4-те „тайни" от config-а — всяка: проблем → показване на живо → какво си тръгват да направят.
3. **60–70 мин** Бонусите (одит 150 €, жива автоматизация, библиотеката) — раздават се НА ЖИВО.
4. **70–85 мин** Офертата: курс 500 € → уебинар цена 397 € с краен срок 48ч → /kurs.
5. **85–90 мин** Q&A. Купилите виждат ъпсела на /kurs/uspeh автоматично.

## Идея „Трейдинг ботове" (отделен проект — НЕ е строен)

Втора фуния за курс „направи си трейдинг бот". **Внимание преди да инвестираме в нея:**
Meta и Google имат много строги правила за реклами на трейдинг/инвестиционни теми —
обещания от типа „автоматични пари от вкъщи" водят до отхвърлени реклами и **блокиран
рекламен акаунт** (риск и за основния бизнес). Ако я правим: рамкираме като
„образование по алгоритмична търговия", без обещания за доходност, с ясен disclaimer,
в отделен рекламен акаунт. Да го обсъдим преди старт.

## Webinar Flow (date-triggered автоматизации)

Зададеш ли `dateISO` в `lib/webinar/config.ts`, cron-ът `/api/cron/webinar-flow`
(на всеки час, vercel.json) сам изпълнява стъпките спрямо датата:

| Стъпка | Кога | Какво праща |
|---|---|---|
| reminder_3d | −3 дни | „какво ще видиш" + подканя към подаръка |
| reminder_1d | −1 ден | напомняне + бонусите са само на живо |
| reminder_1h | −1 час | „започваме след час" + Zoom линк |
| followup_offer | +3 часа | оферта курс 397 € (48ч валидност) |
| lastchance_24h | +27 часа | последен шанс за уебинар цената |

Дедуп: activity `webinar_email_{stage}` на контакт (виждаш всяко изпращане в CRM
профила). Късно записалите НЕ получават изтекли стъпки. Логика: `lib/webinar/flow.ts`.

## Презентацията за уебинара

`/webinar/prezentacia` (noindex, не е линкната публично) — 40 слайда ≈ 90 минути по
„Perfect Webinar" скрипта от архива (`D:\Работа Про Маркетинг Онлайн\Други\Уебинар Фуния`).
Управление: → / ← / клик, **N = бележки за говорителя** (пълен скрипт какво казваш на
всеки слайд), F = цял екран, таймер горе вдясно. Споделяш този екран в Zoom.
